# GEOX Web v1

Vite + React + TypeScript + uPlot.

## Start

```bash
cd GEOX/apps/web
npm install
npm run dev
```

The dev server proxies `/api/*` to `http://127.0.0.1:3001`.

## Notes

- UI only displays backend facts (`samples`, `gaps`, `overlays`).
- No frontend overlay generation, no conclusions.
