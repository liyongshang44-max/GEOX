# H54.5 — Task Linkage

## Purpose

H54.5 proves that a ready operation plan can be projected into a task through the existing from-operation-plan route.

This task does not implement a new task system.

## Task-line position

```text
H53.2 WaterStressState
→ H53.3 Forecast
→ H53.3 ScenarioSet
→ H53.4 RecommendationCandidate
→ H54.0 Control Connector Preflight
→ H54.1 Recommendation Approval Request Connector Gate
→ H54.2 Actionable Irrigation Approval Adapter
→ H54.3 Approval Decision Linkage
→ H54.4 Operation Plan Linkage
→ H54.5 Task Linkage
```

## Runtime proof

```text
operation_plan_v1 READY
+ operation_plan_index_v1 READY
+ approval_request_v1 APPROVED transition
→ POST /api/v1/actions/task/from-operation-plan
→ operator_operation_plan_task_projection_submission_v1
→ ao_act_task_v0
→ operation_plan_index_v1.act_task_id updated
```

## Boundary

Allowed:

- temporary probe `operation_plan_v1`
- temporary probe `approval_request_v1` APPROVED transition
- `operator_operation_plan_task_projection_submission_v1`
- `ao_act_task_v0`
- `operation_plan_index_v1.act_task_id` update

Forbidden:

- `operation_plan_transition_v1`
- `dispatch_v1`
- `ao_act_receipt_v1`
- `as_executed_record_v1`
- `evidence_artifact_v1`
- `acceptance_result_v1`

- `roi_ledger_v1`
- `field_memory_v1`
