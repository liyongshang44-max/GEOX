# H53.2 — WaterStressState Derivation from H53.1 Sensing-Only Inputs

## Purpose

Generate the first derived state from the H53.1 sensing-only lane.

H53.2 turns scoped soil moisture sensing input and weather forecast input into a deterministic `water_state_estimate_v1` fact and a `water_state_estimate_index_v1` readback row.

## Scope

Allowed writes:

- `water_state_estimate_v1` fact with source `H53_2_WATER_STRESS_STATE_DERIVATION_V1`.
- `water_state_estimate_index_v1` row for the same derived state, when the table exists.

Allowed reads:

- H53.1 `C8_SENSING_ONLY_SEED_V1` facts.
- `telemetry_index_v1`.
- `device_observation_index_v1`.
- `soil_moisture_sensing_window_index_v1`.
- `weather_forecast_index_v1` or `weather_forecast_fact_v1`.

## Hard boundaries

H53.2 does not create:

- `root_zone_soil_water_state_v1`
- `root_zone_soil_water_forecast_v1`
- `irrigation_scenario_set_v1`
- `decision_recommendation_v1`
- `approval_request_v1`
- `approval_decision_v1`
- `operation_plan_v1`
- `ao_act_task_v0`
- `ao_act_receipt_v1`
- `as_executed_record_v1`
- `evidence_artifact_v1`
- `acceptance_result_v1`
- `water_response_verification_v1`
- `roi_ledger_v1`
- `field_memory_v1`

## Anti-contamination rule

H53.2 must not treat historical `water_state_estimate_index_v1` rows as current H53.2 output.

The Operator Evidence Twin read model may mark WaterStressState as AVAILABLE only when it finds a fact with:

- `source = H53_2_WATER_STRESS_STATE_DERIVATION_V1`
- `type = water_state_estimate_v1`
- `payload.derivation_version = h53.2.v1`
- matching `tenant_id / project_id / group_id / field_id`

## Current deterministic rule

The first derivation is deliberately simple and explainable:

- `soil_moisture_percent < 16` → `SEVERE_DEFICIT`
- `16 <= soil_moisture_percent < 22` → `MODERATE_DEFICIT`
- `22 <= soil_moisture_percent < 30` → `ADEQUATE`
- `soil_moisture_percent >= 30` → `WET`

Weather contributes a dry-demand signal:

- `water_balance_72h_mm = rainfall_forecast_mm_72h - et0_mm_72h`

The rule does not create recommendations or irrigation amounts.
