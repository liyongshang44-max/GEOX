# H53.3 — Forecast + Scenario from H53.2 WaterStressState

## Purpose

H53.3 generates the first forecast and scenario layer from the H53.2 WaterStressState output.

It turns the H53.2 `water_state_estimate_v1` into:

- `root_zone_soil_water_forecast_v1`
- `root_zone_soil_water_forecast_index_v1` when the table exists
- `irrigation_scenario_set_v1`
- `irrigation_scenario_set_index_v1` when the table exists

## Scope

Allowed reads:

- H53.2 `water_state_estimate_v1` facts with:
  - `source = H53_2_WATER_STRESS_STATE_DERIVATION_V1`
  - `payload.derivation_version = h53.2.v1`
  - matching tenant/project/group/field scope

Allowed writes:

- `root_zone_soil_water_forecast_v1` fact with source `H53_3_FORECAST_SCENARIO_DERIVATION_V1`
- `root_zone_soil_water_forecast_index_v1` row, if the table exists
- `irrigation_scenario_set_v1` fact with source `H53_3_FORECAST_SCENARIO_DERIVATION_V1`
- `irrigation_scenario_set_index_v1` row, if the table exists

## Hard boundaries

H53.3 does not create:

- `decision_recommendation_v1`
- `approval_request_v1`
- `approval_decision_v1`
- `operation_plan_v1`
- `ao_act_task_v0`
- `ao_act_receipt_v1`
- `as_executed_record_v1`
- `evidence_artifact_v1`
- `acceptance_result_v1`
- `water_response_verification_v1`
- `roi_ledger_v1`
- `field_memory_v1`

## Scenario boundary

H53.3 creates comparable scenario options only. It does not select a best option and does not mark any option as a recommendation.

Required options:

- `NO_ACTION`
- `IRRIGATE_10MM_DAY0`
- `IRRIGATE_20MM_DAY0`
- `DELAY_3_DAYS_THEN_IRRIGATE_20MM`

## Deterministic model

This version uses a simple water-balance projection:

- initial available water fraction is derived from H53.2 soil moisture percent.
- daily water balance uses 72-hour rainfall and ET0 divided across the first three forecast days.
- scenario options apply irrigation water as a bounded increase to available water fraction.

This is not an agronomic prescription and not an irrigation recommendation.
