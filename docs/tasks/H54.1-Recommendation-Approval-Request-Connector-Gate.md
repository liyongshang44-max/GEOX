# H54.1 — Recommendation → Approval Request Connector Gate

## Purpose

H54.1 adds an executable connector gate between the H53.4 water-stress recommendation candidate and the existing approval request control chain.

This task does not create a new approval system. It decides whether the current recommendation is eligible to enter the existing approval request path.

## Task-line position

```text
H53.2 WaterStressState
→ H53.3 Forecast
→ H53.3 ScenarioSet
→ H53.4 RecommendationCandidate
→ H54.0 Control Connector Preflight
→ H54.1 Recommendation → Approval Request Connector Gate
```

recommendation_id = rec_h53_4_0fa3d7158a6582f5

suggested_action.action_type = NO_ACTION

This recommendation must not create an approval request, operation plan, AO-ACT task, acceptance, verification, ROI, or Field Memory.

## Gate lanes

### NO_ACTION lane

When the H53.4 recommendation has `NO_ACTION` or non-positive irrigation amount, H54.1 returns:

```text
gate_status = CONTROL_CHAIN_NOT_REQUIRED
gate_lane = NO_ACTION
approval_request_created = false
approval_decision_created = false
operation_plan_created = false
ao_act_task_created = false
acceptance_created = false
verification_created = false
roi_created = false
field_memory_created = false
```

### Actionable irrigation lane

When a future H53.4 recommendation has a positive irrigation action, H54.1 must not directly execute it. It should be adapted to the existing approval request builder contract in a later implementation step.

For this PR, the actionable lane is detected but not submitted:

```text
gate_status = ACTIONABLE_IRRIGATION_REQUIRES_APPROVAL_REQUEST_ADAPTER
gate_lane = ACTIONABLE_IRRIGATION
approval_request_created = false
```

## Existing approval request builder contract

The existing approval request builder expects a legacy/actionable recommendation shape:

- `status = CANDIDATE`
- `source = ROOT_ZONE_SCENARIO_SELECTION`
- `recommendation_kind = IRRIGATION_CANDIDATE_FROM_SCENARIO`
- non-empty `source_option_id`
- evidence refs present
- `proposed_action.action_type` in `IRRIGATE | DELAYED_IRRIGATION`
- `proposed_action.total_irrigation_mm > 0`
- `proposed_action.total_effective_irrigation_mm >= 0`

The current H53.4 C8 demo recommendation is not that shape and is intentionally blocked from approval request creation.

## Boundary

Allowed:

- read the latest H53.4 `decision_recommendation_v1` fact
- classify the recommendation into a connector gate lane
- return a deterministic JSON result

Forbidden:

- write `approval_request_v1`
- write `approval_decision_v1`
- write `operation_plan_v1`
- write `ao_act_task_v0`
- write `ao_act_receipt_v1`
- write `as_executed_record_v1`
- write `evidence_artifact_v1`
- write `acceptance_result_v1`
- write `water_response_verification_v1`
- write `roi_ledger_v1`
- write `field_memory_v1`
