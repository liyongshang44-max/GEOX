# H54.3 — Approval Decision Linkage

## Purpose

H54.3 proves that a recommendation-derived approval request created from the H54.2 actionable irrigation adapter can be decided by the existing approval decision route without creating an operation plan or AO-ACT task.

This task does not implement a new approval decision service. It links the H54.2 recommendation-to-approval-request proof to the already-existing approval decision route.

## Task-line position

```text
H53.2 WaterStressState
→ H53.3 Forecast
→ H53.3 ScenarioSet
→ H53.4 RecommendationCandidate
→ H54.0 Control Connector Preflight
→ H54.1 Recommendation Approval Request Connector Gate
→ H54.2 Actionable Irrigation Approval Adapter
→ H54.3 Approval Decision Linkage
```

## Runtime proof

The acceptance creates only temporary probe records:

```text
H53.4-like positive irrigation recommendation
→ H54.2 adapter
→ existing approval request route
→ approval_request_v1 PENDING
→ existing approval decision route
→ approval_decision_v1 APPROVED
→ approval_request_v1 APPROVED transition
```

## Existing routes used

```text
POST /api/v1/operator/recommendations/:recommendation_id/request-approval
POST /api/v1/operator/approval-requests/:request_id/decision
```

## Allowed probe writes

- temporary probe `decision_recommendation_v1`
- `operator_recommendation_approval_request_submission_v1`
- `approval_request_v1` PENDING
- `operator_recommendation_approval_decision_submission_v1`
- `approval_decision_v1`
- `approval_request_v1` APPROVED transition

All probe facts must be cleaned up before the acceptance exits.

## Forbidden writes

- `operation_plan_v1`
- `operation_plan_transition_v1`
- `ao_act_task_v0`
- `ao_act_receipt_v1`
- `as_executed_record_v1`
- `evidence_artifact_v1`
- `acceptance_result_v1`
- `water_response_verification_v1`
- `roi_ledger_v1`
- `field_memory_v1`

## Boundary

H54.3 stops after approval decision. It must not create operation plans and must not call AO-ACT.

The next step, if accepted, is H54.4:

```text
approval_decision_v1 APPROVED
→ existing create-operation-plan route
→ operation_plan_v1 CREATED
```

H54.4 must still not create AO-ACT tasks.
